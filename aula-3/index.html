Anotações sobre manipulação de strings em C

#include <string.h> strcpy( ) strncpy( ) strcmp( ) strncmp( ) strlen( ) strcat( ) strncat( ) Feedback

É o que faz tudo funcionar

É uma das bibliotecas principais em C aprendida no ultimo smestre onde é necessaria para ter acesso a todos os outros metodos para manipular strings! Sem ela, não é possível usar nenhuma das funções abaixo!


strcpy( ) - Copiar string

Função usada para copiar o conteúdo de uma string para outra

Sintaxe: strcpy(destino, origem);

Exemplo:

char nome[20];
strcpy(nome, "João");  // nome agora contém "João"

strncpy( ) - Copiar com limite

Similar ao strcpy( ), mas permite limitar o número de caracteres copiados

Sintaxe:strncpy(destino, origem, n);

Exemplo:

char nome[20];
strncpy(nome, "João Vitor", 4);  // copia só "João"

strcmp( ) - Comparar strings

Função que compara duas strings e retorna:

  • 0 se as strings são iguais
  • < 0 se a primeira é menor
  • > 0 se a primeira é maior

Sintaxe: strcmp(string1, string2);

Exemplo:

if (strcmp(senha, "1234") == 0) {
printf("Acesso permitido");
}

strncmp( ) - Comparar com limite

Compara apenas os primeiros n caracteres de duas strings

Sintaxe: strncmp(string1, string2, n);

Exemplo:

// Compara só os 4 primeiros caracteres
if (strncmp(nome, "João", 4) == 0) {
printf("Começa com João");
}

strlen( ) - Tamanho da string

Retorna o numero de caracteres de uma string

Sintaxe: strlen(string);

Exemplo:

char nome[] = "Vitor";
int tam = strlen(nome);  // tam = 5

strcat( ) - Concatenar strings

Junta (concatena) duas strings, adicionando o conteúdo da origem ao final do destino

Sintaxe: strcat(destino, origem);

Exemplo:

char saudacao[50] = "Olá, ";
strcat(saudacao, "João");  // saudacao = "Olá, João"

strncat( ) - Concatenar com limite

Junta apenas os primeiros n caracteres da origem ao destino.

Sintaxe: strncat(destino, origem, n);

Exemplo:

char nome[50] = "João";
strncat(nome, " Vitor Santos", 6);  // adiciona só " Vitor"
